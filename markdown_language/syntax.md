# 마크다운 문법 정리
[참조 링크](https://heropy.blog/2017/09/30/markdown/)

- 제목: \<h1> ~ \<h6>까지 사용가능
# h1 #
## h2 ##
### h3 ###
#### h4 ####
##### h5 #####
###### h6 ######
h1 (===)
===
h2 (---)
---

- 강조: 두껍게, 이텔릭체, 밑줄, 취소선
**\**두껍게\****
__\__두껍게\____
*\*이텔릭체\**
_\_이텔릭체\__
<u>\<u>밑줄\</u></u>
~~취소선\~~~~

- 목록 : 순서있는 목록(1. , 2., 3. ...), 순서없는 목록(-, +, *)
1. 목록1
  - 목록1 - 서브1
  - 목록1 - 서브2
2. 목록2
  1. 목록2 - 서브1
  2. 목록2 - 서브2

- 링크 : 소괄호 사용(직접 링크 입력), 대괄호 사용(참조 링크 입력)
소괄호 사용 : [소괄호](https://www.google.co.kr/)
